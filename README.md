# Revenue AI Copilot

Revenue AI Copilot is an AI-powered assistant designed for hotel Revenue Managers.

The project combines:

- Revenue Management knowledge
- Retrieval-Augmented Generation (RAG)
- PDF document processing
- Large Language Models (LLMs)

## Current Capabilities

- Load Revenue Management PDFs
- Extract and chunk documents
- Search relevant content using MinSearch
- Generate answers with Groq LLMs

## Example Questions

- What is RevPAR?
- Why is market segmentation important?
- What are the benefits of dynamic pricing?
- What is channel management?

## Tech Stack

- Python
- Jupyter Notebook
- Groq
- MinSearch
- RAG
